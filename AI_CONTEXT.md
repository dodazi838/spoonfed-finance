# 🥄 떠먹여주는 금융경제 - AI 개발 컨텍스트 (AI_CONTEXT.md)

이 문서는 다른 PC 환경이나 새로운 AI 인스턴스(Cursor, Gemini, Copilot 등)가 본 프로젝트의 아키텍처와 최근 변경 사항을 빠르게 이해할 수 있도록 작성된 요약 가이드입니다.

## 📌 1. 프로젝트 개요
- **목적:** 대학생 및 일반인을 위한 공공 경제 리포트(한국은행 등) AI 요약 분석 서비스
- **기술 스택:** Next.js (App Router), React, CSS Modules, Recharts (차트), React-Markdown (에디토리얼 UI)
- **AI 모델:** Google Gemini 2.5 Flash (`@google/generative-ai` SDK)

## 🏗️ 2. 아키텍처 및 핵심 파일
- `src/app/page.tsx`: 프론트엔드 메인 UI. (파일 업로드 -> 목차 추출 -> 목차 선택 -> 챕터별 분석)
- `src/app/api/analyze/route.ts`: 파일 업로드 및 전체 요약/목차(챕터) 추출 API
- `src/app/api/analyze-chapter/route.ts`: 선택된 개별 챕터에 대한 정밀 심층 분석 및 차트 데이터 추출 API
- `src/components/ReportResult.tsx`: AI가 반환한 JSON 데이터를 렌더링 (Markdown 텍스트 + Recharts)
- `src/components/ReportResult.module.css`: 결과 화면의 프리미엄 에디토리얼 스타일(글래스모피즘, 인용구 등)

## 🧠 3. 최근 AI 프롬프트 및 UI 고도화 내역
1. **프리미엄 에디토리얼 UI 적용 (Rich Markdown)**
   - 초기 버전의 촌스러운 색상(빨강/파랑 텍스트, 노란 형광펜)을 전면 제거.
   - **인사이트 박스:** 챕터 내 가장 중요한 통찰(1~2문장)은 마크다운 인용구(`>`)로 추출하여, CSS를 통해 좌측 파란 선이 들어간 세련된 요약 박스(`blockquote`)로 렌더링함.
   - 핵심 키워드 강조는 모노톤 볼드체(`**`)로만 처리하며, 형광펜(`<mark>`)은 텍스트 하단에 은은한 파란색 밑줄(Underline)이 그어지도록 CSS(`border-bottom`)로 오버라이드 됨.
2. **짧은 리포트 편의성 (Select All)**
   - `isShortReport` 속성이 `true`일 경우, 프론트엔드(`page.tsx`)에 "전체 선택하기" 버튼이 활성화되어 사용자가 모든 챕터를 원클릭으로 선택 가능.

## ⚠️ 4. 구글 API Rate Limit (429 Error) 및 백오프(Backoff) 히스토리
- **원인:** 구글 무료 티어의 극악한 한도 (1분당 15회 요청 / 1분당 100만 토큰 / 일 1,500회 요청). 사용자가 챕터를 3개 이상 선택하여 병렬 요청 시, 1분 토큰 한도(TPM)를 즉각 초과하여 429 에러 발생.
- **히스토리:** 이를 극복하고자 백엔드에 '5회 자동 재시도(retries=5, delay=15초)' 로직을 추가했으나, 1분 밴에 걸렸을 때 백그라운드에서 헛발질 요청을 수십 번 날리게 되어 **'일 1,500회 한도(RPD)'마저 순식간에 방전시키는 심각한 부작용** 발생.
- **현재 해결책:** 일일 쿼터 방전을 막기 위해 재시도 횟수를 `2회`로 대폭 축소함. 무료 티어 운용 시, **긴 리포트는 챕터를 1~2개씩만 선택**해야 1분 토큰 밴(100만 토큰)을 피할 수 있음을 사용자에게 안내해야 함.
